# Web Geliştirme; Javascript Proje Bilgisi

Katılımcılar eğitim programı çerçevesinde elde ettikleri bilgileri hayallerindeki fikirler ile harmanlayıp kullanarak bir uygulama projesi hazırlar. Kavramları bütüncül kullanmaları için aşağıdaki yönergede gösterilen proje adımları takip edilir. Amaç modern web dünyasına girişi hızlandırmak ve eğitimin pekiştirilmesidir.

## Proje Uygulama Adımları

* Modern Javascript kütüphanelerinden biri seçilir.   
  * ReactJS  
  * VueJS  
  * AngularJS  
  * Jquery  
  * ve diğerleri

  \[Netlify veya muadili ile yayına alınabilir bir çerçeve seç.\]

* Seçilen kütüphane veya çerçevesi kurulumu yapar.  
* Kurulumu seçtiği bir IDE ile açar.  
* Proje dosya ağaç yapısında Components, Pages, Interfaces isimlerinde klasörler açar.  
* Tailwind CSS ve ya Bootstrap 5 veya Pure CSS projede kullanır.  
* Eğitimde işlenen veya TODO App benzeri uygulama baz alınarak.  
  * Javascript çerçevesi kullanarak proje geliştirmesi yapar.  
    * Bir adet Ekle işlemi,  
    * Bir adet Listeleme işlemi,  
    * Bir adet Güncelleme işlemi,  
    * Bir adet Silme işlemi gerçekleştir.  
* En az 1 adet proje ekran görüntüsü ekle.  
* Github hesabında public bir alana proje klasörlerini yükle.  
* Github linkini, proje teslim formu içerisinde paylaş.  
* Netlify veya muadili ile yayına al ve proje tesliminde paylaş.

## Proje Çıktıları

- [ ] Html temellerini uygular, geliştirme yapar.  
- [ ] CSS temellerini uygular, kütüphaneler kurar ve geliştirmeler yapar.  
- [ ] Javascript temellerini uygular, kütüphaneler kurar ve geliştirme yapar.  
- [ ] ReactJS vb. kütüphane temellerini uygular, geliştirme yapar.  
- [ ] Github hesabına proje yükleyerek deneyim kazanır.  
- [ ] Gerçek bir frontend projesi başlamış olur.
